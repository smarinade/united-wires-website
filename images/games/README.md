# Game Images

Add game cover images here. Recommended specs:

- **Format**: JPG or PNG
- **Dimensions**: 800x450px (16:9 aspect ratio)
- **File size**: Under 200KB each
- **Quality**: High quality screenshots or official artwork

## Where to Get Images

1. **Steam Store Pages** - Header images
2. **Official Game Websites** - Press kits/media sections  
3. **SteamGridDB** - https://www.steamgriddb.com/
4. **Google Images** - Search "[game name] cover art" or "wallpaper"

## Tips

- Use high-quality official artwork when possible
- Crop to 16:9 aspect ratio for consistency
- Optimize images before adding (use tinypng.com)
- Test on the site - emoji placeholders show if image is missing
